# The Librarian

## Description

The librarian is the server for the book tracker 9000.

## Technology Considerations

#### Technology Used
- JavaScript/TypeScript
- [NodeJS](https://nodejs.org/en)
- [NestJS](https://nestjs.com/)
- [Prisma](https://www.prisma.io)
- PostgresSQL

## Running the project

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
